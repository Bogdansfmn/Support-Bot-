🎯 Описание проекта
Support Bot — это микросервис для автоматизации первой линии технической поддержки. Он принимает входящие сообщения от пользователей через вебхуки, анализирует их, определяет категорию и приоритет обращения, подбирает готовый ответ из базы знаний или передаёт запрос оператору с полным контекстом диалога.

🚀 Основная идея
Сервис работает как промежуточное звено между системой поддержки (СпросиИИ) и операторами. Он автоматически обрабатывает типовые запросы, снижая нагрузку на сотрудников, а сложные вопросы передаёт специалистам с уже подготовленным контекстом.

Установка и запуск
🐍 Локальный запуск
bash
# 1. Клонируйте репозиторий
git clone https://github.com/anB154519/support-bot.git
cd support-bot

# 2. Создайте виртуальное окружение
python -m venv .venv
source .venv/bin/activate  # Linux/Mac
# или
.venv\Scripts\activate     # Windows

# 3. Установите зависимости
pip install -r requirements.txt

# 4. Запустите сервер
python main.py
📦 Установка зависимостей
bash
pip install -r requirements.txt
🚀 Запуск сервера
bash
# Через Python
python main.py

# Через Uvicorn (с автоперезагрузкой)
uvicorn main:app --reload --host 0.0.0.0 --port 8000
После запуска сервер будет доступен по адресу: http://localhost:8000
